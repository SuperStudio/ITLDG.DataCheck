# 数据校验

`ITLDG.DataCheck` 为数据校验核心,提供常见的串口校验:`BCC`,`CheckSum`,`LRC`,`CRC`等等

`Example` 为调用该核心实现的校验推算

`PluginExample` 插件示例,可以自己编写插件来为推算提供更多可能
